# Mini Social Media Platform

A mini social media web application developed as a **first-year college team project** using core web technologies.  
The application allows users to create posts, view them in a feed, and retain data even after page refresh using browser LocalStorage.

## ğŸ§  Application Flow

1. User enters text in the input field
2. On clicking the **Post** button:
   - JavaScript captures the input
   - Data is stored in a posts array
   - Posts array is saved in LocalStorage
   - Feed is re-rendered using DOM manipulation
3. On page reload:
   - Data is retrieved from LocalStorage
   - Posts are displayed again automatically
